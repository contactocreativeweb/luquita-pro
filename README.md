# 💱 Luquita PRO - Convertidor BCV en Vivo

**Luquita PRO** es una aplicación web progresiva (PWA) diseñada para proporcionar conversiones de divisas en tiempo real utilizando las tasas oficiales del Banco Central de Venezuela (BCV). Con una interfaz moderna, rápida y optimizada para dispositivos móviles, Luquita PRO facilita el cálculo de tasas para Dólares ($), Euros (€) y más.

![Luquita PRO Preview](https://luquita-app.vercel.app/logo-512.png)

## 🚀 Características Principales

- **Tasas en Tiempo Real**: Actualización constante de las tasas oficiales del BCV.
- **Convertidor Multimoneda**: Soporte para USD (Dólar), EUR (Euro) y cálculos sugeridos automáticos.
- **Experiencia Premium**: Diseño moderno con soporte para Modo Oscuro y Modo Claro.
- **PWA (Progressive Web App)**: Instálala en tu dispositivo móvil o escritorio para acceso instantáneo, incluso sin conexión a internet constante.
- **Compartir Fácil**: Botón integrado para compartir la aplicación rápidamente con otros usuarios.
- **Optimización Total**: Carga ultrarrápida y consumo mínimo de datos.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: HTML5, Vanilla JavaScript.
- **Estilos**: Vanilla CSS (Custom Design System).
- **PWA**: Service Workers para soporte offline y manifiesto de aplicación web.
- **Deployment**: Configurado para despliegue automático en [Vercel](https://vercel.com).
- **Fuentes**: Google Fonts (Outfit).

## 📦 Instalación y Uso Local

Si deseas ejecutar este proyecto localmente para desarrollo:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/contactocreativeweb/luquita-pro.git
   ```

2. Entra al directorio del proyecto:
   ```bash
   cd luquita-pro
   ```

3. Abre el archivo `index.html` en tu navegador o usa un servidor local como Live Server.

## 📱 Instalación como App

### En iOS (iPhone/iPad):
1. Abre Luquita PRO en Safari.
2. Toca el botón **Compartir** (cuadrado con flecha hacia arriba).
3. Selecciona **"Agregar a inicio"**.

### En Android / Chrome:
1. Abre Luquita PRO en Chrome.
2. Toca los tres puntos en la esquina superior derecha.
3. Selecciona **"Instalar aplicación"**.
